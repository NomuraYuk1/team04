1. クラス名: Player      
属性（変数）一覧:   
float x, y
float speed
boolean isTrapped
int webEscapeCount
メソッド一覧:   
void move()
void display()
void checkTrap()
各属性、各メソッドの内容に関するメモ:  
x, y: プレイヤーの現在位置を管理する座標。   
speed: プレイヤーの移動速度。通常時、水たまり進入時などで数値が変化する。   
Trap: 蜘蛛の巣に捕まって足止めされているかどうかを判定するフラグ。   
EscapeCount: 蜘蛛の巣から脱出するために、特定のキーを何回押したかを記録するカウンター。   
move(): キーボードの入力に応じて、プレイヤーの座標 x, y を更新して移動させる処理。   
display(): プレイヤーのキャラクターを画面上に描画する処理。   
checkTrap(): 蜘蛛の巣に捕まっている間の移動制限や、キー連打による脱出判定を管理する処理。  

2. クラス名: Puddle      
属性（変数）一覧:   
float x, y
float w, h
float slowRate
メソッド一覧:   
void display()
boolean hitTest(Player p)
各属性、各メソッドの内容に関するメモ:   
x, y: 水たまりを配置する位置の座標。   
w, h: 水たまりの領域の横幅と縦幅。   
slowRate移動速度を低下させる倍率。   
display(): 水たまりの範囲を青色などの指定した色で画面に描画する処理。  
hitTest(Player p): プレイヤーが水たまりの領域内にいるかを判定し、重なっている場合はプレイヤーの移動速度を低下させる処理。   

3. クラス名: SpiderWeb     
属性（変数）一覧:   
float x, 
float w, 
hint requiredTaps
メソッド一覧:   
void display(boolean hitTest(Player p)
各属性、各メソッドの内容に関するメモ:   
x, y: 蜘蛛の巣を配置する位置の座標。   
w, h: 蜘蛛の巣の領域の横幅と縦幅。   
requiredTaps: 足止めを解除するために必要なキーの連打回数。 
display(): 蜘蛛の巣の範囲を画面に描画する処理。   
hitTest(Player p): プレイヤーが蜘蛛の巣に接触したかを判定し、接触した瞬間にプレイヤーの isTrapped を真（true）にする処理。   

4. クラス名: Gate      
属性（変数）一覧:   
float x, y
float w, 
hboolean 
Open
int intervalTime
int lastSwitchTime
メソッド一覧:   
void update()
void display()
boolean isPassable()
各属性、各メソッドの内容に関するメモ:   
x, y: ゲートを配置する位置の座標。   
w, h: ゲートの横幅と縦幅（壁としてのサイズ）。   
Open: ゲートが開いているか（通行可能か）、閉じているか（通行不可か）を表すフラグ。   
intervalTime: 自動で開閉を切り替える場合の制限時間（ミリ秒）。   
lastSwitchTime: 最後に開閉状態が切り替わった時間を記録する変数。   
update(): 一定時間の経過、または特定のキーが押されたことを検知して isOpen の状態を反転（チェンジ）させる処理。   display(): ゲートの開閉状態に応じて、色や形を変えて画面に描画する処理。   
Passable(): 現在プレイヤーや敵がこのゲートを通行できるかどうかを外部（メインプログラムなど）に返す処理。

メソッド一覧:

各属性、各メソッドの内容に関するメモ:
