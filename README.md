#### 要件定義

##### 概要

　パックマンのような見下ろし型のゲーム

・自機はマップに点在する宝を制限時間内に回収することが目的

・マップには敵が存在。触れるとゲームオーバー
・マップにはアイテムが存在し、使うことで自機が有利になる



##### 各担当

###### 自機・敵

・自機の移動システムの構築

　・見下ろし型のため、4方向への移動が必須

　・アイテムを使うキー設定も必要

・敵の移動システムの構築

###### 

###### アイテム

・各種アイテムの実装

　・移動速度UP、敵への攻撃・デバフ、ランダム効果

###### 

###### マップ

・自機や敵が壁に埋まることがないよう、当たり判定等の実装

・移動速度Down、無敵地帯など、特殊なエリア実装



###### UI

・制限時間、敵数、宝の数、スコア、ゲームクリア、ゲームオーバー時の表示を担当

　・余裕があればタイトル画面等も？
